# SpecKit 规约编程使用指南

本指南主要向项目的协同开发者、AI 编程助手（如 Antigravity / Cursor / Windsurf 等）说明，如何使用本项目集成的 `SpecKit 10步法Workflow` 功能，实现完全标准化的软件开发。

---

## 🧐 什么是 SpecKit?

SpecKit (规约编程) 是一个集成了自动需求提取、自动系统架构定义、智能 TDD（测试驱动开发）逻辑的 Workflow（工作流）库。
该机制将**软件开发标准化成了不可跳过的 10 个精确步骤**。

它的核心目的是要求您的 AI 助理：
- **拒绝直接敲代码**：任何新代码落地前必须生成“功能拆解”、“系统架构”、“工时估算”。
- **不跑偏不遗漏**：依据 `constitution.md` (项目规约) 生成的内容能够让 AI 少走几万字的弯路。
- **天然自带文档库**：开发结束，直接顺带产出可阅读、可归档的项目全套交付报告。

---

## 🛠️ 如何唤起工作流?

在具有 Workflow 支持的 IDE Agent 对话框（或 Composer）里，支持**在开头敲入 `/speckit.xxx` 命令**来直接让 AI 开始干活。

> **核心原则：从 ① 走到 ⑩，每一步 AI 都会在项目里的 `docs/` 和 `.specify/` 生成对应的前置报告！**

### 1. 启动新功能开发：获取需求 (①)
假设老板让你：“给目前的语雀 MCP 服务增加一个‘读取语雀原文’（`read_yuque_note`）的工具能力”。
不要直接让 AI 开始写代码，请在对话框输入：

```text
/speckit.demand 给现在的 MCP Server 增加读取语雀正文（read_yuque_note）的功能，参数传 book_id 和 doc_id。
```

> **发生什么？** AI 会新建一个新功能分支/文件夹，深入理清你的需求并反问你，最终将其沉淀为 `docs/demand/` 下的规格书。

### 2. 进行架构与设计 (②、③、④)

需求确认后，依次在后续对话中运行以下指令：

```text
/speckit.product       # （步骤2：如果涉及 UI/逻辑图，这步会画出脑图）
/speckit.architecture  # （步骤3：AI 会查阅 MCP SDK，告诉你怎么做更稳健，并画出接口表）
/speckit.plan          # （步骤4：AI 估算开发需要多久，分析核心难点）
```

> **发生什么？** 这三步走完后，你的 AI 已经对整个功能的难点（比如鉴权是否改变、Node.js 跨域）了然于胸。

### 3. 分配任务并正式写代码！ (⑤、⑥)

```text
/speckit.tasks         # （步骤5：讲刚才庞大功能拆解为一个个 Todo 小票）
/speckit.implement     # （步骤6：让 AI 开始读取小票，一行行修改代码！！）
```

> **发生什么？** 在 `implement` 阶段，AI 知道此时的约束条件是什么（比如本项目规约里“禁用 console.log”的要求），这极大减小了 AI 修改代码的翻车率。

### 4. 测试以确保稳定性 (⑦、⑧、⑨)

```text
/speckit.unittest      # （步骤7：强制 AI 根据刚刚写的代码补充 Vitest 测试脚本）
/speckit.testcase      # （步骤8：生成一份给人看的用例文档）
/speckit.testreport    # （步骤9：输出本次功能开发的跑测成功率）
```

### 5. 总结与归档 (⑩ 以及辅助命令)

代码测试通过并且主干被合并后，告诉 AI：

```text
/speckit.report        # （步骤10：生成项目交付文档，用于给领导汇报/复盘）
/speckit.archive       # 📦 打包刚刚这一轮的所有记忆文档，清空大脑，准备下一个新需求
```

## 🌟 补充超强命令：需求中途变卦怎么办？

如果已经按着之前的文档做到了一半，产品经理突然说：*“诶，参数改一下，别传 doc_id 了，传 slug （文档链接里的短名称）。”*

不用从头来，直接打：
```text
/speckit.update 将获取文章的参数从 doc_id 变更为 slug，其他不变。
```

> AI 会**自动分析影响了之前生成的哪些规划图和任务清单**，并仅精确修改关联的部分。

--- 

> **开发者自定规约约定**：
> 如需调整项目整体的底线要求（譬如：从此以后的代码必须写全 JSDoc），请让 AI 去修改 `.specify/memory/constitution.md` 即可约束住它的潜意识。
